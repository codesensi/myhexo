idea2021.2.3试用方案

发表于2021-10-23|更新于2021-10-23|教程工具
|字数总计:252|阅读时长:1分钟|阅读量:

安装

下载安装最新版，下载地址：官网

临时试用

2021.2.3版本开始，idea试用需要登录账号或者输入key

方式一

输入Activation code

key获取地址：热心大佬提供的key

将上图红框内的内容复制到如下图位置

点击Activate

方式二

官网注册账号，此处不做具体演示

无限试用

通过大佬开发的插件无限试用

下载

下载地址：ide-eval-resetter
访问密码：2462

使用

将下载的压缩包解压

idea新建任意项目，进入项目，将解压后的jar包拖入编辑区，出现如下提示

点击Accept，提示插件安装成功

点击工具栏Help-Eval Reset

勾选Auto reset before per restart之后点击Reset

弹出的对话框中选择Yes重启idea

到此idea无限试用成功，之后每次重启idea就会重置试用时间为30天

理论上此无限试用方案适用于idea全家桶
